Trend.In

Trend.In was born druing a 24 hours hackathon (Unihack 2015) and it's essentially a webapp which can be used to add tags (each tag can hold data such as links, messages, javascript, url, etc) to any image on imgur.

MovieFinder

A lite-weighted react app (with the flux architecture) which helps you find movies using omdbapi.com API

Newspod

Newspod is another project that was created during the Code Fest hackathon. Newspod is a modern news site, created using the hearld sun api and text-voice assistance so you can listen to any type on news on the go.

NotifyME

NotifyME is a web-app which is built on top of Goggle Maps (using NodeJS, Postgres SQL, Express, Google Maps API, Javascript, CSS) which consists the disruptions data (planning permit, road closures, public events, level crossing projects) of Melbourne city. Its purpose is to assist users in reducing their travel time from point A to B.



Workthrive (Startup)

Work Thrive is an innovative solution that aims to breakdown employee goals to their core, and with the magic of analytics, provide quantitative data that better tracks and informs managers of their employees growing skill sets and performance.
